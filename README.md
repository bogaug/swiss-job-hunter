# 🇨🇭 Swiss Job Hunter

**A Streamlit app to search for jobs in Switzerland, generate AI-powered cover letters, and track your applications.**

**📱 Now available as a mobile PWA — deploy for free and install on Android!**

## ✨ Features

| Feature | Description |
|---|---|
| 🔍 **Job Search** | Searches via Adzuna API (free) for Swiss jobs by region |
| 🤖 **AI Cover Letter** | Generates personalized cover letters using Groq (free LLM API) |
| 📧 **Email Sending** | Sends applications automatically via SMTP (Gmail, Outlook…) |
| 📊 **Dashboard** | Tracks all applications with statuses, history timeline, and charts |
| 🌍 **Multilingual** | Cover letters in French, Italian, English, or German |
| 📱 **Mobile PWA** | Install as native app on Android (or any device) |
| ⬇️ **CSV Export** | Export your applications for external analysis |

## 🚀 Deployment (FREE)

### Option 1: Deploy to Render.com (Recommended - 2 minutes)

**Full step-by-step guide:** [DEPLOY_RENDER.md](DEPLOY_RENDER.md)

**Quick start:**
1. Go to [render.com](https://render.com) → Sign up (free, no credit card)
2. **New** → **Web Service** → Connect `bogaug/swiss-job-hunter`
3. Click **Create** (auto-detects config)
4. Add environment variables:
   - `GROQ_API_KEY` → Get free at [console.groq.com](https://console.groq.com)
   - `ADZUNA_APP_ID` + `ADZUNA_APP_KEY` → Get free at [developer.adzuna.com](https://developer.adzuna.com)

Your app will be live at: `https://swiss-job-hunter-XXXX.onrender.com`

**Install on Android:**
1. Open Chrome on your phone
2. Go to your Render URL
3. Menu ⋮ → "Add to Home screen"
4. ✅ Native app icon appears on home!

### Option 2: Run locally (desktop + mobile)

**Desktop (Streamlit):**
```bash
git clone https://github.com/bogaug/swiss-job-hunter.git
cd swiss-job-hunter
pip install -r requirements.txt
streamlit run app.py
```

**Mobile PWA (same network):**
```bash
# Start both Streamlit + FastAPI
./start-pwa.sh  # Mac/Linux
start-pwa.bat   # Windows
```

Script shows the URL to open on your phone (e.g., `http://192.168.1.100:8000`)

### 1. Clone the repository

```bash
git clone https://github.com/bogaug/swiss-job-hunter.git
cd swiss-job-hunter
```

### 2. Create a virtual environment and install dependencies

```bash
python -m venv .venv
source .venv/bin/activate   # macOS / Linux
.venv\Scripts\activate      # Windows

pip install -r requirements.txt
```

### 3. Configure environment variables

```bash
cp .env.example .env
# Edit .env with your preferred editor
```

Required variables:

| Variable | Description |
|---|---|
| `GROQ_API_KEY` | Get yours free at [console.groq.com](https://console.groq.com) |
| `CANDIDATE_NAME` | Your full name |
| `CANDIDATE_EMAIL` | Your email address |
| `SMTP_USER` | Sender email address |
| `SMTP_PASSWORD` | SMTP App Password |

> **Gmail users:** generate an App Password at [myaccount.google.com/apppasswords](https://myaccount.google.com/apppasswords) (requires 2FA to be enabled).

### 4. Run the app

```bash
streamlit run app.py
```

The app will open automatically at `http://localhost:8501`.

## 🤖 AI Cover Letter Generation

Letters are generated by **Groq** (free API) using open-source LLMs, following Swiss epistolary conventions.

**Get your free API key at [console.groq.com](https://console.groq.com) — no credit card required.**

**Available models:**
- `llama-3.3-70b-versatile` — maximum quality (recommended)
- `llama-3.1-8b-instant` — fast and lightweight
- `mixtral-8x7b-32768` — alternative with large context window

---

## 🔍 Job Search

Uses **Adzuna API** (free, 250 requests/day) to search Swiss job market.

**Without API key:** Demo mode with 5 example jobs to test the interface.

**With API key:** Real-time job search across Switzerland by region.

Get your free credentials at [developer.adzuna.com](https://developer.adzuna.com)

---

## 📊 Application Tracking

Each application stores:
- Job title, company, location, URL, HR email
- Current status with full history timeline
- Generated cover letter
- Submission date and notes

**Available statuses:** Sent → Under Review → Phone Interview → In-Person Interview → Assessment → Offer Received → Accepted / Rejected / Withdrawn

---

## 🔒 Privacy

- The SQLite database is stored locally in `/data/` (gitignored)
- SMTP credentials and API keys stay in `.env` (gitignored)
- No data is sent to external servers other than the Anthropic API

## 📱 Deploy gratuito per smartphone (Android/iOS)

L'app include una **Progressive Web App (PWA)** installabile su qualsiasi smartphone.

### 🆓 Opzione 1: Render.com (Raccomandato — sempre gratis)

1. Vai su **[render.com](https://render.com)** → Sign up con GitHub
2. **New** → **Web Service**
3. Seleziona il repo `bogaug/swiss-job-hunter`
4. Render rileva automaticamente `render.yaml` → **Create Web Service**
5. Ottieni URL pubblico: `https://swiss-job-hunter.onrender.com`

**Sul tuo smartphone:**
- Apri **Chrome** (Android) o **Safari** (iOS)
- Vai all'URL ricevuto
- **Android**: Menu ⋮ → "Installa app"
- **iOS**: Condividi → "Aggiungi a Home"
- Icona 🇨🇭 sulla schermata home → funziona offline come app nativa!

**Piano gratuito:** 750 ore/mese (= sempre disponibile). L'app va in sleep dopo 15 min inattività, si risveglia in ~30 secondi al primo accesso.
